RSA 公钥-私钥加密

纯前端实现 · 密钥与运算均在你的浏览器内，不发起任何网络请求

私钥 (priv.json)
公钥 (pub.json)

题意流程：给定私钥 + 明文 → 输出公钥 + 密文。选 private 用私钥 d 加密；选 public 用公钥 e 加密（标准）。

公钥 (从私钥派生)
密文 (base64)

解密时 mode 与 padding 须与加密时一致：private 密文用公钥解，public 密文用私钥解。

明文

PSS 签名：用私钥对消息签名，输出签名。这是「私钥运算」的安全形态。

签名 (base64)

PSS 验签：用公钥验证消息与签名是否匹配。

实现与 rsa.py 一一对应 · OAEP/PSS 符合 RFC 8017 (PKCS#1 v2.2)
